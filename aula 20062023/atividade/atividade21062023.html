ATIVIDADE - TAGS DE TEXTO

Nesta atividade, você aprenderá a utilizar tags de texto para formatar o conteúdo em uma pagina web.

A seguir, temos como exemplo de trecho de codigo em HTML para enfatizar o texto:

<em> Texto Enfatizado <em>

Experimente utilzar <b> que deixa em NEGRITO, ou outras Tags que podemos usar para deixar nossa frase em italico, como <em>, por exemplo.

Outras tags que podemos usar são:

- <strong> para negrito e relevancia ao texto

- <u>, para texto sublinhado.

- <sub>, para textos subscritos.

Entre outras, para ver como elas irão alterar nossos textos.
