Atividade - Tags de Texto

Nesta atividade, você aprenderá a utilizar tags de texto para formatar o conteúdo em uma página web

A seguir, temos um exemplo de trecho de código em HTML para enfatizar um texto.

<em>Texto enfatizado</em>

<em>Lorem ipsum dolor sit amet consectetur adipisicing elit. Sint ratione dolorum et provident quidem, nam ex unde reiciendis eum vitae fugiat error repellat facilis nisi, nostrum eaque quae voluptate recusandae?</em>

Experimente utilizar outras tags de texto, como <strong>, <u>, <sub>, <sup>, entre outras, para ver como elas alteram a aparência do texto.


Exemplo <strong> Negrito (Aparência + Mais relevante): Lorem ipsum

Exemplo <u> (Sublinhado): Lorem, ipsum

Exemplo <sub> (Subescrito): Lorem ipsum

Exemplo <sup> (Sobreescrito): Lorem ipsum
